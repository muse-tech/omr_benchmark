# PDF Files

Augmented PDF files for the benchmark can be downloaded from our HuggingFace dataset:

[![HuggingFace Dataset](https://img.shields.io/badge/ðŸ¤—%20HuggingFace-Dataset-yellow)](https://huggingface.co/datasets/musegroup/omr_benchmark)

**Download:** [musegroup/omr_benchmark](https://huggingface.co/datasets/musegroup/omr_benchmark)

Each PDF is generated from public domain scores and augmented to simulate real-world scans:
- Ink blobs / stains
- Scratches / wear
- Crumpled or textured paper
- Rotation / skew
- Other visual noise
